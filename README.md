# Sticky Header Anchor Fixer

Fixes anchor links on sites with sticky headers by offsetting scroll position so target elements are visible.

## Features
- Set header offset (px)
- Toggle smooth scroll
- Apply to internal links only
- Accessibility-friendly: focuses target after scroll

## Installation
1. Upload plugin folder to `/wp-content/plugins/`
2. Activate plugin
3. Visit Settings → Sticky Anchor Fixer and set header offset

## Development
- PHP compatible with 7.4+ (update as you target)
- Vanilla JS, no external libs
